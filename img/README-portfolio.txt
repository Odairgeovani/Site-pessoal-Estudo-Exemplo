IMAGENS DO PORTFÓLIO

Este arquivo explica os placeholders necessários para as imagens do portfólio.

Em um ambiente de produção, você deve adicionar as seguintes imagens à pasta img/:

1. portfolio1.jpg - Site Corporativo
2. portfolio2.jpg - Loja Virtual
3. portfolio3.jpg - Landing Page
4. portfolio4.jpg - Blog Profissional
5. portfolio5.jpg - Página de Produto
6. portfolio6.jpg - Marketplace

Cada imagem deve ter:
- Proporção de aspecto consistente (preferencialmente 16:9 ou 4:3)
- Resolução mínima de 800x600 pixels
- Formato JPG ou WEBP otimizado para web
- Representação visual clara do tipo de projeto

Além disso, você precisará de imagens para os depoimentos de clientes:
1. client1.jpg - Ana Silva
2. client2.jpg - Carlos Mendes
3. client3.jpg - Mariana Costa

As imagens de clientes devem ser:
- Formato quadrado
- Resolução de 200x200 pixels
- Preferencialmente com fundo neutro

Substitua todas estas imagens por capturas de tela reais de seus projetos e fotos reais de seus clientes (com permissão) antes de publicar o site.
